О проекте

«photoStoked» - это фото-банк/фотосток, т. е. Интернет-платформа, предоставляющая возможность продажи и покупки лицензий на использование изображений. Проект создавался и был защищён как дипломный.

Ссылка на репозиторий: код проекта

Для разработки проекта были использованы фреймворки Laravel и Vue.js. Был использован модуль, установленный на Vue.js owl-carousel.

Главная

На главной странице пользователю видны все новейшие работы, популярные авторы и слайдер с самыми популярными работами.

нажмите, чтобы включить демонстрацию

Каталог

В каталоге пользователь может найти подходящий ему материал несколькими способами. Он может выбрать одну из существующих меток. Метки указываются при отправлении работы пользователем. И автоматически заносятся в базу данных и «приписываются» работе при её публикации на сайт.

нажмите, чтобы включить демонстрацию

Поиск также можно осуществить при помощи поисковой строки. При таком поиске находятся работы, чьи метки полностью или частично ответствуют содержанию поисковой строки.

нажмите, чтобы включить демонстрацию

Все работы или уже отобранные по метке или в соответствии с поисковым словом можно отфильтровать, уточнив формат или тип изображения.

нажмите, чтобы включить демонстрацию

Авторизация и регистрация

Чтобы воспользоваться всеми возможностями сайта, пользователю нужно зарегистрироваться или авторизоваться. Валидация полей осуществляется на стороне сервера. При регистрации проверяются уникальность имени и почты, прочность пароля и является ли фото профиля изображением.

нажмите, чтобы включить демонстрацию

При авторизации пользователя его переносит на его личную страницу.

нажмите, чтобы включить демонстрацию

Редактирование данных о пользователе

Пользователь может отредактировать свои личные данные и добавить информацию о себе. Новые данные тоже проходят валидацию.

нажмите, чтобы включить демонстрацию

Отправка работы

Пользователь может отправить свою работу на проверку. Она не появится на сайте, пока администрация не одобрит её.

нажмите, чтобы включить демонстрацию

Страница другого пользователя

При переходе на страницу другого пользователя у авторизованного пользователя появляется возможность добавить его в избранное.

нажмите, чтобы включить демонстрацию

На других пользователей можно отправить жалобу.

нажмите, чтобы включить демонстрацию

Взаимодействия с работами

Работы можно добавлять в понравившиеся.

нажмите, чтобы включить демонстрацию

Пользователь имеет возможность приобрести определённое количество «покупок». Покупкой работы считается первое её скачивание. Тогда она добавляется в раздел «Мои покупки». После работу можно скачать неограниченное количество раз. За каждую покупку автору работы начисляются деньги на счёт сайта, которые потом можно вывести.

нажмите, чтобы включить демонстрацию

Пользователь может создать коллекцию. Коллекции позволяют сохранять и объединять работы.

нажмите, чтобы включить демонстрацию

При добавлении первой работы в коллекцию она станет её обложкой. Если работа, которая была обложкой, будет удалена из коллекции, то её роль займёт другая.

нажмите, чтобы включить демонстрацию

Уведомления

На сайте присутствуют уведомления. Пользователь получает их, когда его работу выкладывают на сайт, его работе ставят лайк, его работу скачивают, когда пользователя добавляют в избранное.

нажмите, чтобы включить демонстрацию

Действия администратора

В панели администратора можно увидеть все отправленные на проверку работы. Их можно скачать для проведения проверки. После указания нужных данных работа выставляется. Администратор вручную загружает только версию изображения с водными знаками, оригинальная версия автоматически переноситься в папку принятых работ.

нажмите, чтобы включить демонстрацию

Работу можно отклонить.

нажмите, чтобы включить демонстрацию

В разделе работы с пользователями отображены все пользователи сайта, кроме администрации. Пользователя можно найти по имени с помощью поисковой строки.

нажмите, чтобы включить демонстрацию

Пользователя можно удалить с сайта. Для этого нужно нажать на кнопку удаления дважды.

нажмите, чтобы включить демонстрацию
